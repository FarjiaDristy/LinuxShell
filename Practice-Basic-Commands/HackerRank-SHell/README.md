# Linux Shell Practice: HackerRank Exercises

This repository contains my daily practice and exercises (learn, certify, and compete) for learning Linux shell commands, mainly drawn from the HackerRank **Shell** domain. It is organized to help me (and others) follow a learning path from beginner to intermediate topics.

## How to use this repo

1. Browse the Markdown files for explanations and example commands.
2. Open scripts in the `exercises/` folder to see working solutions.
3. Run them in a Linux shell to practice.

## What this repo covers

This repo collects solutions and notes for Linux‑shell problems including the following subdomains:

- Bash scripting  
- Text processing (grep / sed / awk)  
- Arrays in Bash  
- Grep / Sed / Awk challenges
